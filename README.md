# ⚽ CopaNaMão

App mobile para torcedores acompanharem o jogo, pedirem lanche e reverem lances sem sair do assento.

## 👥 Integrantes
- Arthur Daniel

## 🎯 Problema Focado
O torcedor perde lances importantes ao se levantar para comprar comida ou se localizar no estádio. O CopaNaMão resolve isso reunindo *mapa, pedido de lanche e replays* em um só lugar.

## 🎨 Justificativa de Design
Cada tela tem uma função única (Heurística 8 de Nielsen — design minimalista). O carrinho exibe o resumo completo antes de confirmar, prevenindo erros (Heurística 5). As abas do cardápio e os filtros de replay permitem navegação rápida sem perder o foco no jogo.

## 🔄 Fluxo do Usuário

*Pedir lanche:* Tela inicial → Pedir Comida → escolher categoria → selecionar item → Carrinho → Finalizar Pedido.

*Ver replay:* Tela inicial → Ver Replays → filtrar por tipo (Gols/Faltas/VAR) → selecionar lance → escolher câmera → Voltar ao vivo.
